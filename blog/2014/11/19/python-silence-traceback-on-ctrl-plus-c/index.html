| Comments

Came across this small little hack while writing my donwloader for NCBI-SRA datasets(Should be on PyPI soon)

What did we just do?

using signal.signal() I set the handler to sys.exit(0) for signal.SIGINT which is translated to a KeyBoardInterrupt exception(invoked when you do a Ctrl+C)
